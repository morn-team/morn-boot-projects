# OperationLog 操作日志

MornBoot提供通用的注解及处理器，完成对操作日志的监听和处理。

Since：v1.0.0

# 必要配置

SpringBootApplication

@EnableNotify // 开启系统通知

# 推荐配置

#开启操作日志默认转换器
morn.operate.default-converter.enabled=true
#开启国际化
morn.translator.enabled=true
#国际化资源文件
spring.messages.basename=morn/operation

# Operate Annotation

使用注解及线程变量描述操作日志

@Component // Spring容器扫描
@OperateGroup("user") // 描述操作模块，此处为用户模块
class TestUserController {

  @OperateAction("add") // 描述操作类型，此处为新增用户
  Map<String, Object> addUser(Map<String, Object> user) {
    OperateArguments.add(user.get("username")); // 记录日志参数
    return user;
  }

  @OperateAction("update") // 描述操作类型，此处为更新用户
  Map<String, Object> updateUser(Map<String, Object> user) {
    OperateArguments.add(user.get("username")); // 记录日志参数
    OperateArguments.add(user.get("password")); // 记录日志参数
    throw new RuntimeException("异常测试"); // 抛出异常后，日志会记录为操作失败
  }
}

# Operation Processor

实现OperationProcessor接口，以处理操作日志。MornBoot会监听操作结果，并完善操作内容。

success：默认true，业务代码抛出异常时，值为：false
content：默认从国际化资源文件中获取，key为log.[group].[action]

注意：必须使用实例注解 (opens new window)，否则Bean Enhance (opens new window)不会扫描该Bean

@Slf4j
@Component
@Objective
public class TestOperationProcessor implements OperationProcessor {

  @Override
  public void accept(Operation operation) {
    log.info(operation);
    // 持久化操作日志
    // ...
  }
}

例一：addUser的操作日志

Operation(success=true, module=user, operator=null, content=新增用户：timely, date=Wed Dec 05 22:48:56 CST 2018)

例二：updateUser的操作日志

Operation(success=false, module=user, operator=null, content=修改用户：rain, date=Wed Dec 05 22:48:56 CST 2018)

国际化资源文件：

log.user.add=新增用户：{0}
log.user.update=修改用户：{0}

# Operation Converter

MornBoot默认使用Translator生成操作内容，你也可以自定义操作日志。

application.properties

#关闭操作日志默认转换器
morn.operate.default-converter.enabled=false

参考以下代码编写新的转换器

注意：必须使用实例注解 (opens new window)，否则Bean Enhance (opens new window)不会扫描该Bean

@Objective
public class DefaultOperationConverter implements OperationConverter {

  /**
   * 翻译器
   */
  private final Translator translator;

  public DefaultOperationConverter(Translator translator) {
    this.translator = translator;
  }

  @Override
  public Operation generic(OperateMeta operateMeta) {
    // 拼接国际化编码
    String messageCode = "log." + operateMeta.getModule() + "." + operateMeta.getName();
    // 生成操作内容
    String content = translator.translate(messageCode, operateMeta.getArguments());
    // 操作时间
    Date date = new Date();
    // 构建操作日志实例
    return Operation.builder().success(operateMeta.isSuccess())
        .module(operateMeta.getModule()).content(content).date(date).build();
  }
}
